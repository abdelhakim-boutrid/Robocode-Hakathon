## Comment tester mes robots ?
Prérequis : Avoir Robocode installé sur votre machine.

Installation :

1-Clonez ce repository.

2-Copiez les fichiers .java (ou compilez les en .class) dans le dossier /robots de votre installation Robocode.

3-Lancement : Ouvrez Robocode, allez dans Battle -> New, et sélectionnez mes robots dans la liste.
